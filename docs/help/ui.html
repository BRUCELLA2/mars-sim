Main Window

The Mars Simulation Project's main window contains the following components:

  1. Menu Bar

  2. Mission Tool

  3. Monitor Tool

  4. Sound Tool

  5. Mars Navigator Tool

  6. Time Tool

  7. Search Tool

  8. Person Info Window

  9. Rover Info Window

  10. Settlement Info Window

  11. Unit Bar
