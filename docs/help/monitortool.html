Monitor Tool

The Monitor tool displays information in a table layout of all the people, vehicles, settlements and missions in the simulation. It also displays historical events and trade goods.

Top Buttons - The Monitor tool has the following top buttons:

  • Pie Chart - Creates a tab with a pie chart of one column from the table.

  • Bar Chart - Creates a tab with a bar chart of multiple columns from the table.

  • Remove Tab - Removes the currently selected custom tab.

  • Center Map On Selected Unit - Centers the map and globe in the Mars Navigator tool on the selected unit.

  • Show Details Dialog - Opens a information window on the selected unit.

  • Open Mission - Opens the selected mission in the mission tool. Only enabled when the mission tab is selected.

  • Change The Displayed Columns - Lets the user modify what information is displayed in the Monitor tool.

  • Filter Events - Lets the user filter what category of events are displayed in the historical events tab.
