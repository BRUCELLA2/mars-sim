landmarks.xml


The landmarks.xml configuration file contains configuration for landmarks on Mars that can be displayed in the user interface. Currently only point landmarks (rather than area landmarks) are supported.

Each "landmark" element has a unique name attribute.

The "longitude" and "latitude" attributes must have the location degrees followed by the direction ("E" or "W" for longitude and "N" or "S" for latitude).
