User Interface

The Mars Simulation Project provides the following visual interface elements at your disposal :

A. Keyboard Shortcuts

B. Tab Bar

    1. Tools on the Main Tab

    • Search Tool
    • Time Tool
    • Monitor Tool
    • Mission Tool
    • Science Tool
    • Resupply Tool

    2. Tools on the Map Tab

    • Mars Navigator Minimap
    • Settlement Map

    3. Tools on the Help Tab

    • Help Browser

C. Unit Info Window on the Main Tab

  • Person Info Window
  • Bot Info Window
  • Vehicle Info Window
  • Settlement Info Window
  • Equipment Info Window

D. Bars, Box and Panels

  • Earth Time Bar : show Earth time in UT
  • Mars Time Bar : show Martian time in UMST, open up Mars Calendar Panel
  • Sound Panel : adjust the sound volume for background and sound effects
  • Speed Panel : adjust the time ratio, view Ticks per sec and UpTime
  • MarsNet Chat Box : connect to settlers, bots, settlement and vehicle

E. Buttons and Toggle Buttons

  • Speed Button
  • Chat Button : open up MarsNet Chat Box
  • Sound Button

F. Toggle Buttons

  • Cache Toggle Button : cache Minimap and Settlement Map
  • Minimap Toggle Button
  • Settlement Map Toggle Button
