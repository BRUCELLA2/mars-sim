Help Browser

The Help Browser is a tool that provides buttons to quickly take you to the inline tutorial, the user guide, the about page as well as the online resources.

The address bar shows you the local path of the html page you are reading or the hyperlink of the website you're visiting.

The '<' button (or backspace) returns you to the page you previously visited.

The 'Go' button reloads the link on the address bar.

The '>' button sends you back to the next page that you have visited before you hit the '<'.

Note: normally, the page you are currently on is saved as the 'next' page for you to returns to whenever you hit '<'.

Link Preview

If your mouse cursor hovers over a hyperlink, it will attempt to print out its local path or http link on the status bar.
