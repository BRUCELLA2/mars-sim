Tutorial - Step 1 : Introduction

Welcome to the Tutorial of Mars Simulation Project (mars-sim).

Through these steps you will become familiar with the built-in tools to view and interact with the simulation.

The tutorial has 12 steps mostly focused on the usage of various Tool Windows.

Expect it to take about 10-15 minutes to complete.

The tutorial is supplemented by a more in-depth User Guide, which explains various concept of mars-sim.

The User Guide is accessible from the Help menu.

You may close this tutorial window at any time and reopen it from the Help menu.
