Tutorial - Step 9 : Settlement Map Tool

The Settlement Map Tool is where you zoom in on a particular settlement - as if you were hovering over it and seeing through the buildings with x-ray vision.

You can select which settlement you want to view from the drop-down list.

The map can be zoomed in and out with the zoom slider, and rotated right or left with the rotate buttons.

The settlement map can moved around by mouse dragging.

The Recenter button will reset the map at the settlement's center, default zoom, and rotation.

The Open Info button opens up the settlement info window for the currently selected settlement.

The Labels button will show a drop down when pressed that allows the user to select which types of labels to display.

The types of labels include daylight tracking, buildings, construction sites, vehicles, people and bots.

Clicking on a person or a bot will display their Info Window and name in white, along with their current activity.
