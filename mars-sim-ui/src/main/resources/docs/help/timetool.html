Time Tool

The Time Tool is your timekeeper on Mars in the simulation.

1. Martian Time : the current Martian time in the format of Orbit-Month-Sol Millisol.

Note : Orbit are the Martian Year and Millisols are between 0 and 1000 up to the 3 decimal places.

2. Martian Calendar : displays a Martian Calendar with the sol of the week/month highlighted.

3. Martian Seasons : shows the current season one would experience on the Northern and southern hemisphere of Mars.

4. Earth Time : shows the corresponding date and time on Earth.

5. Simulation Uptime : displays how long mars-sim has been running.

6. Tick Per Second (TPS) : shows how fast mars-sim updates the changes.

Note : TPS is essentially the same as a more common programming construct called "Frames Per Second" (FPS) in designing the game loop for mars-sim.

e.g. if TPS/FPS is 25, then for each second, the simulation is updated 25 times.

7. Pause Button : click to toggle between pause and resume the simulation or hit the 'ESC' button once on your keyboard anytime.

8. Real to Sim Time Ratio : shows the ratio of the real time to the simulation time.

By default, it is set to the ratio of 1 second of real time to 8 minutes and 20 seconds of simulation time.

You may speed up (or slow down) the simulation by dragging the handle to the right (or to the left).

The Martian calendar used in this simulation is based on Shaun Moss's "Areosynchronous Calendar".
http://pweb.jps.net/~gangale3/moss/Virtual_Mars/Calendar.asp

The Areosynchronous Calendar is heavily derived from Tom Gangale's Darian Calendar.
http://pweb.jps.net/~tgangale/mars/mst/darian.htm

The Martian year is referred to as an "orbit". It is 668.5921 Martian days ("Sols") long. In the calendar, about half of the orbits are 668 Sols long, and the other half are "leap-orbits" with 669 Sols. See the Areosynchronous Calendar for complete details on leap-orbits.

The month names are based on Frans Blok's "The Rotterdam System".
http://www.geocities.com/fra_nl/rotmonth.html

There are seven sols in a Martian week with the following week names:

  1. Heliosol
  2. Neriosol
  3. Libersol
  4. Terrasol
  5. Venusol
  6. Mercusol
  7. Jovisol

Every Martian month has 4 weeks. On months with 27 Sols, the last week only has six Sols and Jovisol is clipped. This allows the first Sol of every month and every orbit to be Heliosol.

The calendar uses metric time for Mars with the Sol broken up into the following common units:

decasol = 1/10 Sol = 2.46 Earth hours
centisol = 1/100 Sol = 14.8 Earth minutes
millisol = 1/1000 Sol = 1.48 Earth minutes

Martian time of the Sol is usually written in three-digit millisol format to three decimal points. ex. from "000.000" to "999.999".

The common timstamp format for Mars date/time is orbit-month-sol:millisols. ex. "17-Adir-03:523.234"

The time of the day is based on Bruce Mackenzie's "Metric Time for Mars".
http://pweb.jps.net/~gangale3/other/mcknzfrm.htm
