Tutorial - Step 8 : Mission Tool

The Mission Tool is where you see all the ongoing missions the settlers are currently having.

In mars-sim, a mission may be generated and completed based upon the need of the settlement without user's intervention.

However, user may manually create new missions and/or modify and/or delete existing ones

The 'Info' tab shows the relevant info of a mission.

The 'Navpoints' tab displays a small map showing various rally points or locations where the mission will take place.
