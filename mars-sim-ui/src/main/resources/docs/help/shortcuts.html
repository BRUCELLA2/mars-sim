Shortcuts

(A). Alt + T : Display shortcuts for all Tool Windows.

F1 : Mars Navigator (Navigator Window)

F2 : Search Tool (Search Window)

F3 : Time Tool (Time Window)

F4 : Monitor Tool (Monitor Window)

F5 : Mission Tool (Mission Window)

F6 : Settlement Tool (Settlement Window)

F7 : Science Tool (Science Window)

F8 : Resupply Tool (Resupply Window)

F9 : Commander Tool (Commander Dashboard)

(See User Interface for more details on each Tool)


(B). Alt + H : Display shortcuts for viewing documents.

Ctrl + H : Open the Home/About page in the Help Tool.

Ctrl + T : Open the Tutorial in the Help Tool.

Ctrl + G : Open the User Guide in the Help Tool.


(C). Alt + F : Display shortcuts for file-related operations.

Ctrl + O : Load a simulation.

Ctrl + L : Load an autosave simulation.

Ctrl + S : Save the simulation with the default filename 'default.sim'.

Ctrl + A : Save As the simulation with the filename of your choice.

Ctrl + X : Exit the Simulation.


(D). Alt + S : Display shortcuts for other settings.

Ctrl + U : Toggle on and off the Unit Bar.

Ctrl + B : Toggle on and off the Tool Bar.

Slider Bar 1 : Control the sound effect volume.

Slider Bar 2 : Control the background music volume.

Ctrl + Up : Turn up the volume of the background music and sound effects (for Windows or Mac OS only).

Ctrl + Down : Turn down the volume of the background music and sound effects (for Windows or Mac OS only).

Ctrl + M : Toggle on and off the background music and sound effects (for Windows or Mac OS only).

Ctrl + E : Toggle on and off the background music and sound effects (for Windows or Mac OS only).
