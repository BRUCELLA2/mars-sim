Shortcuts

ESC : Pause/resume the simulation

F1 : Help Browser

F2 : Search Tool

F3 : Time Tool

F4 : Monitor Tool

F5 : Mission Tool

F6 : Science Tool

F7 : Resupply Tool

Ctrl + A : Save As (the filename of your choice)

Ctrl + S : Save the simulation by the default filename 'default.sim'

Ctrl + X : Exit the Simulation (3 choices : 'Save & Exit', 'Exit', 'Back to Sim')

Ctrl + N : Toggle on and off Notification Panel popping up on the left/right bottom corner.

Ctrl + F : Toggle between the full-screen mode and the window mode.

Ctrl + T : Toggle between the mud orange and snow blue theme (for Windows or Mac OS only).

Ctrl + M : Toggle on and off the background music and sound effects (for Windows or Mac OS only).

Ctrl + Q : Fire a quotation popup on the top right corner.

Ctrl + Up : Turn up the volume of the background music and sound effects (for Windows or Mac OS only).

Ctrl + Down : Turn down the volume of the background music and sound effects (for Windows or Mac OS only).

(See User Interface for more details on each Tool)
