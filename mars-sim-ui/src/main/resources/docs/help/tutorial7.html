Tutorial - Step 7 : Monitor Tool

The Monitor Tool displays the crucial, real-time data for mars-sim in the format of a table with rows and columns.

There are 9 categories of data accessible by the Monitor Tool.

These data set (each shown in a separate tabs) are Bots, Crops, Events, Food, Goods, Missions, People, Settlements and Vehicles

You may examine each of them by clicking on its corresponding tab.
