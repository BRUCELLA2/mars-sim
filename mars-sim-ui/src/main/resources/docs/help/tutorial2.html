Tutorial - Step 2 : The Main Simulation Window

The Main Simulation Window is the large window which forms the base of the simulation.

It houses the Menu Bar on the top, the Desktop (one with a brownish background texture) in the middle and the Status Bar at the bottom.

The Desktop is where all the Tool Windows and Unit Windows are displayed.

Upon start of a new simulation, you will see that the Tutorial has already been opened.
