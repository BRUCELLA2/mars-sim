Tutorial - Step 2 : The Main Simulation Window

The Main Simulation Window is the overall window area upon which the simulation is running.

In full-screen mode, you will see the Tab Bar near the top with the desktop area (one with a brownish background texture) in the middle

In window mode, you will see the addition of the title bar above the main window.

Normally, the desktop area displays the content of the selected tab.

Upon start of a new simulation, you will see that it defaults to open the Help tab where the Help browser is located at.
