Tutorial - Step 3 : Tabs and Shortcuts

Tabs

The Tab bar is located near the top of the main window with a total of 7 tabs available. User may switch to any single tab at any time.

The Main tab : various Unit Info Windows and certain Tool Windows.

The Map tab : the Settlement Map and the Mars Navigator Minimap.

The Monitor tab : the Monitor Tool.

The Mission tab : the Mission Tool.

The Resupply tab : the Resupply Tool.

The Science tab : the Science Tool.

(See Tabs for more details on each tab

Shortcuts

Ctrl + A : Save As (the filename of your choice)

Ctrl + S : Save the simulation by the default filename 'default.sim'

Ctrl + X : Exit the Simulation

Ctrl + N : Toggle on and off Notification Panel popping up on the left/right bottom corner.

Ctrl + F : Toggle between the full-screen mode and the window mode.

Ctrl + T : Toggle between the mud orange and snow blue theme.

Ctrl + M : Toggle on and off the background music and sound effects.

Ctrl + Q : Fire a quotation popup on the top right corner.

Ctrl + Up : Turn up the volume of the background music and sound effects.

Ctrl + Down : Turn down the volume of the background music and sound effects.

(See Tabs for displaying all the shortcuts in future.

F1 : Help Browser

F2 : Search Tool

F3 : Time Tool
