Tabs

The Tab bar is located near the top of the main window with a total of 7 tabs available. User may switch to any single tab at any time.

The Main tab is where the various types of Unit Info Windows are displayed as well as certain Tool Windows.

The Map tab houses Settlement Map as well as the Mars Navigator Minimap.

The Monitor tab is where the Monitor Tool is found. Certain type of settlement data may be aggregated and viewable via graphs and charts using the Monitor Tool.

The Mission tab houses the Mission Tool which are essential for resourcing, planning and interacting with various type of missions.

The Resupply tab is where the Resupply Tool is located. It's the tool to order supplies and parts from Earth for expanding existing settlements or staging new settlements on Mars.

The Science tab houses the Science Tool that provides insights into the scientific achievement of the settlers.
