Site Constructions

A. To put together a building, a construction site needs to go through the following 3 stages :

  1. Laying the site foundation
  2. Framing
  3. Constructing the building

In general, one must first build a foundation. Upon finishing the foundation, one may build a corresponding frame on top of it. When the frame is done, one may construct a particular building over the frame.

B. Twelve Available Foundations :

  1. Surface foundation 3m x 2m
  2. Surface foundation 5m x 5m
  3. Surface foundation 10m x 5m
  4. Surface foundation Variable Size
  5. Sub-surface foundation 5m x 5m x 3m
  6. Sub-surface foundation 5m x 10m x 3m
  7. Sub-surface foundation 10m x 10m x 3m
  8. Sub-surface foundation variable size
  9. Ramped sub-surface foundation 15m x 18m x 5m
  10. Shallow borehole drilling site
  11. Deep borehole drilling site
  12. Unprepared surface foundation
